# Coconut — Brand Voice Reference

**Date:** 2026-05-06
**Source:** Copy analysis across homepage, about, features, MTD landing page, pricing page.

---

## Tone Adjectives

1. **Friendly** — conversational, warm, uses "we" and "you" liberally. Reads like a knowledgeable friend, not a software vendor.
2. **Reassuring** — constantly reduces anxiety around tax, HMRC, and compliance. "No more stress", "worry-free", "it's easier than you think".
3. **Simple** — deliberately avoids complexity. Explains things in plain English and flags that as a feature. "No need to be an accounting expert."
4. **Encouraging** — gently pushes users toward action without pressure. "Get started today", "Sort it today!", "See for yourself."
5. **Informal-professional** — casual enough to say "driving you nuts?" and "a breeze", but never slangy or flippant. Stays credible.

---

## Vocabulary Level

**Casual-professional.** Roughly a Flesch score of 70-80. Short sentences, everyday words, minimal jargon. Technical terms (MTD, EOPS, SA103) are always explained in context. The audience is assumed to have zero accounting knowledge.

## Don't Words

| Avoided word/concept | What they use instead |
|---|---|
| Accounting software (standalone) | "Bookkeeping and tax app", "self employed accounting software" — always qualified. |
| Powerful / robust / enterprise | Never used. Positioning is approachable, not powerful. |
| Leverage / utilise / optimize | Plain English alternatives only. |
| Complicated / complex | Only used to describe the old way ("no more complicated spreadsheets"). |
| Client (for end users) | "You", "sole traders", "community". "Client" reserved for accountant-facing copy. |
| Platform (marketing sense) | Used once on the about page in a vision statement, but not in product copy. |
| Cutting-edge / innovative | Never used. The brand avoids tech-startup language. |
| Solopreneur / freelancer | "Sole trader" is the consistent term. "Freelancer" does not appear. |
| Dashboard | Not used. They say "summary", "overview", "real-time profit overview". |
| Onboard / onboarding | "Get set up", "get started", "get going". |

## Sentence Style

- **Short sentences.** Typical range: 8-18 words. Rarely exceeds 25.
- **Imperative leads.** CTAs and feature intros start with verbs: "Snap photos", "Send and track", "See your profit".
- **Question hooks.** Headlines frequently use questions: "Are you ready for Making Tax Digital?", "Are your taxes driving you nuts?"
- **Negation-to-positive pattern.** Names the pain, then removes it: "No more tax stress", "Never lose a receipt again", "No more complicated spreadsheets."
- **Second person dominant.** "You" and "your" throughout. First person "we" used for the company, never for the user.
- **Lists over paragraphs.** Feature benefits are almost always bulleted, not prose.
- **Colloquial interjections.** "a breeze", "a real chore", "driving you nuts", "pronto" — one per section maximum, never stacked.

## Voice Consistency Notes

**Strengths:**
- Very consistent tone across all five pages. The friendly-reassuring register holds.
- Pain-point framing is effective and repeated without feeling forced.
- "Plain English" positioning is genuine — the copy practices what it preaches.

**Weaknesses / Inconsistencies:**
- The about page is noticeably thinner and more generic than other pages. The vision statement ("enable this economy to boom") uses startup-pitch language that doesn't match the rest of the voice.
- "Super easy" is overused — appears on homepage hero, features intro, and multiple feature cards. Risks sounding unsubstantiated.
- Some CTAs are generic ("Read more", "More information") and don't match the otherwise action-oriented voice.
- Pricing page hero has a typo: "Making Tax Digial" (missing 't').
