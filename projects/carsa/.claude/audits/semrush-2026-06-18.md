## Notable Changes Since May

### Improved
- **Multiple H1 tags:** 42 → 2. Chatbot H1 fix landed on most pages. Remaining: /car-care/drive-away-car-insurance and /used-cars/promotions/free-1-year-warranty-2025.
- **Pages crawled:** 5,398 → 5,484. Site growing with inventory.

### Regressed
- **Structured data errors:** 0 → 1,235. This is the biggest regression. Affects make pages (/used-cars/make/*), model pages, fuel filter pages, and /car-care/car-paint-interior-protection. Likely a schema deployment issue on CMS template pages.
- **Slow page load:** 3,540 → 4,032 (+14%). More pages now flagged.
- **Duplicate titles:** 1,862 → 1,950 (+5%). Growing with new inventory.
- **Duplicate meta descriptions:** 44 → 50.

### Unchanged
- Zero 5xx server errors
- Zero redirect chains/loops
- Valid robots.txt and sitemap.xml structure
- Valid SSL certificate
- No DNS issues
- HTTP links in terms pages still unfixed (3 pages)

## Pages with Issues — Detail

### Title tag missing or empty (3 pages) — NEW
- /stores/bca-bedford
- /stores/bca-leeds
- /stores/mansbridge

All three are new store pages added since May. Missing both title and meta description. Also have external 403 errors.

### Broken internal links (10 pages)
- /used-cars/models/bmw-i4 — model page linking to removed VDPs
- 9 VDPs with broken internal links (likely cross-referencing sold vehicles)

### 4xx errors (38 pages)
All are VDPs under /vehicles/used/. These are sold cars whose pages have been removed but are still linked from elsewhere on the site.

### Pages with only one internal link (93 pages)
Mix of:
- Blog posts with poor internal linking (benefits-of-part-exchanging, carsa-at-the-british-motor-show-2024, carsa-stock-vans-for-sale, etc.)
- Niche model pages (chery-tiggo-8, gwm-funky-cat, kgm-torres)
- VDPs that have lost their category page links

### Nofollow in internal links (18 pages)
- 7 fuel filter pages (/used-cars/fuel/*) — each has 2 nofollow internal links
- 8 sell-car store pages — each has 1 nofollow internal link
- /sell-car/part-exchange — 1 nofollow internal link

### Structured data errors (1,235 pages)
Predominantly:
- /used-cars/make/* pages (all makes)
- /used-cars/models/* pages
- /used-cars/fuel/* pages
- /car-care/car-paint-interior-protection

**Action required:** Validate schema on a sample make/model page using Google Rich Results Test. Likely a required field missing from the CMS template schema.
