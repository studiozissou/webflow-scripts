## Key Findings

### Broken External Links (5,291)
Highest-count issue. These are outbound links pointing to external URLs that return errors. Likely patterns:
- Dealer network links that have changed
- Third-party service URLs that have moved
- Social media profile URL changes

**Action:** Export full list, categorise by domain, fix or remove in bulk.

### Slow Page Load (3,540 pages)
Predominantly VDPs and store pages. Causes:
- Heavy JS bundles (VWO, n8n Chat, JetBoost, GSAP)
- Mapbox GL on store pages
- Unoptimised vehicle images
- CDN scripts loading with @latest (no version pinning)

### Duplicate Titles (1,862)
VDP title template likely generating titles that are too similar when make/model/year combinations overlap. Template restructuring needed.

### Duplicate Meta Descriptions (44)
Up from 2 in the previous scan. CMS template pages sharing the same description field value. Each page needs a unique, CMS-bound description.
